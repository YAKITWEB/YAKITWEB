# YakıtWeb Backend

Kurumsal YakıtWeb uygulaması için Express tabanlı backend API.

## Başlatmak için

1. Gerekli paketleri yükleyin:
   ```
npm install
   ```
2. Geliştirme modunda başlatmak için:
   ```
npm run dev
   ```
3. Normal başlatmak için:
   ```
npm start
   ```

## Sağlık Kontrolü

- `GET /api/health` endpointi ile backend'in çalışıp çalışmadığını kontrol edebilirsiniz.
